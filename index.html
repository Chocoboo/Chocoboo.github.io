正题

闲着无聊，发现A站页面右下角的AC娘还蛮有意思的。首先呢，每摸五十次她就会换一个姿势，不仅如此，两个姿势之后，他还会测试你小左小右的手速，反正我是最少300+。

在这里呢，在下不才，想写一个面向青铜组选手的教程，其实也就是有几点需要注意的。

关于点击事件

每次点击的时候，会有一个提示点击次数的小框向上渐变运动，运动可以用css3中的@keyframes如下

@-webkit-keyframes click{
    50%{
        -webkit-transform:translateY(-15px);
        opacity: 1;
    }
    100%{
        -webkit-transform:translateY(-30px);
        opacity: 0;
    }
}

再单独设置一个.an的style

.an{animation: 1.5s click;}

注意这里是单独，因为该方法是通过每次点击创建一个Li标签的，同时给Li添加”an”的className

function createLi(){
    removeChildren(oUl);  //清除Li
    var aLi = document.createElement('li');
    aLi.innerHTML = '+1&nbsp;&nbsp;Click!';
    aLi.className = 'click an';
    oUl.appendChild(aLi);                        
}

完了别忘了移除这些Li元素，这里是方法：

function removeChildren(pnode){    
    var childs=pnode.childNodes;    
    for(var i=0;i<childs.length-1;i++){    
    pnode.removeChild(childs.item(i));    
    }    
}

# JSX

简单点来说即是Html和Javascript可以混写：

var Hello = React.createClass({
    render: function(){
        return (
            <div>
                <p>Hello,React.</p>
            </div>
        )
    }
})

这里需要注意：

1.与原 HTML 写法不同的属性：class -> className、for -> htmlFor
2.一定要写关闭标签，自闭和标签也别忘记。如<input />
3.render 方法返回的必须是一个顶层标签
